# 評価

完成度はrailsのほうが上。

ただ、利用できる人が多くて、動かせる環境が多いPHPで動くのが強み。

ログの表示などで、自前の実装がそれなりにある。

railsと違い、vite,tailwindが最初から使えるのもメリット。

このように、laravelの開発者の、技術選択のセンスはいい。

## 新しい技術

新しい技術の中には、有益なものもある。

具体的には、vite, island(vue), tailwindなど。

ただ、避けておいたほうが無難なものもある。

具体的には、`spa-test`参照。

新しい技術は、すべて否定するのでもなく、すべて肯定するのでもなく、取捨選択していかないと、いろいろとしんどいことになる。
